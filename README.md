# visual-servo
Image-based visual servoing base on ROS2
# To Run
## Starting gazebo simulation environment
```bash
ros2 launch z1_description gazebo.launch.py
```
## Running visual servo control program
```bash
ros2 run z1_control z1_vs_velocity.py
```
